Of Repute - A simple resume API.

Site Layout

./
This help documentation
./api
The actual resume data api is located. This data is served with a 'Content-Type' of 'application/json;utf-8'.

API Data Access Description

The entire JSON document is navigable, so each array key/object property that is listed is available for direct access.

Examples:

./api
./api/experiance
./api/skills/0
./api/contact
